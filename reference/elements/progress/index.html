Progress

Attributes

The progress element inherits all the attributes of BaseElement

In addition, it has the following attributes:

AttributeRequiredDescriptionTypeExample
valueYesValue of the progress Resolvable <number>
paramsNoAdditional HTML attributes Resolvable < HTMLProgressElement >

Example

<script lang="ts">
  import { Base, Form } from '$lib';

  const form = Base.newForm([
    {
      type: 'progress',
      value: 0.5,
    },
  ]);
</script>

<Form {form} />
